### Theme Generator

Generates SCSS file from json file

#### JSON File format 
```
  colorModel        : string, set to "hsl" to generate custom colors in HSL format
  classes           : string[], create color classes. Possible values are "bg", "text", "accent", "outline"  
  generateRawValues : boolean, all variables will be replace by real color values in the final output
  tailwind          :
  {
    colors: string[], additional tailwind color variables to include in the output
                      "all" for all variables  
  } ,
  "theme" :[
    {
      name  : string, name of theme color variable
      var   : string, name of tailwind color on which this variable is defined as
    },
    or
    {
      name : string, name of theme color variable
      H: {
        base : integer (0 - 359), base HUE value, this is for variable-500,
        min  : integer (0 - 359), min HUE value, this is for variable-50,
        max  : integer (0 - 359), max HUE value, this is for variable-900
      },
      S: {
        base : integer (0 - 100), base SATURATION value, this is for variable-500,
        min  : integer (0 - 100), min SATURATION value, this is for variable-50,
        max  : integer (0 - 100), max SATURATION value, this is for variable-900
      },
      L: {
        base : integer (0 - 100), base LIGHTNESS value, this is for variable-500,
        min  : integer (0 - 100), min LIGHTNESS value, this is for variable-50,
        max  : integer (0 - 100), max LIGHTNESS value, this is for variable-900
      }
    }
  ]
}
```

#### Example JSON file

```
{
"colorModel" : "hsl",
  "classes" : ["bg", "text"],
  "generateRawValues": true,
  "tailwind" : {
    "colors": ["indigo", "rose", "slate"]
  } ,
  "theme" :[
    {
      "name": "primary",
      "var": "violet"
    },
    {
      "name" : "secondary",
      "H": {
        "base" : 100,
        "min" : 20,
        "max" : 250
      },
      "S": {
        "base" : 80,
        "min" : 50,
        "max" : 99
      },
      "L": {
        "base" : 50,
        "min" : 25,
        "max" : 80
      }
    }
  ]
}
```
